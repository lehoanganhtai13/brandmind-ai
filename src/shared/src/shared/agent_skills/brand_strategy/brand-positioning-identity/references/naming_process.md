# Brand Naming Process — 6 Steps

## When to Use
- **new_brand**: Run full 6-step process
- **refresh/repositioning**: Usually skip (set skipped=True with reason)
- **full_rebrand**: Run unless strong equity in current name

## Step 1: Choose Naming Approach

Select the primary approach based on positioning:
- **Descriptive**: directly describes what the brand does (e.g., "The Coffee House")
- **Suggestive**: hints at benefits or experience (e.g., "Highlands")
- **Abstract**: coined or invented words (e.g., "Phúc Long")
- **Experiential**: evokes an emotion or experience
- **Heritage/Origin**: references place, founder, or tradition

## Step 2: Generate 10-15 Candidates

Brainstorm names using the chosen approach. Consider:
- Vietnamese market compatibility (pronunciation, meaning)
- English accessibility (international appeal if relevant)
- Category distinctiveness (avoid generic category terms)
- Emotional resonance with target audience

## Step 3: Linguistic Screening

Screen each candidate for:
- Vietnamese pronunciation ease and meaning associations
- English pronunciation and potential negative meanings
- Cross-language compatibility
- Spelling simplicity and memorability
- No unintended meanings in relevant languages

Eliminate candidates with linguistic issues. Target 8-10 surviving candidates.

## Step 4: Availability Checks

For surviving candidates, check:
- **Domain**: .com, .vn availability (use `search_web` to verify)
- **Social handles**: Instagram, Facebook, TikTok (use `search_web`)
- **Trademark**: basic search in Vietnam IP database

Record availability status per candidate. Eliminate or flag candidates with critical conflicts.

## Step 5: Evaluate Top 5 Against Keller's 6 Criteria

Score each of the top 5 candidates (1-5 per criterion):

| Criterion | What It Measures |
|-----------|-----------------|
| **Memorable** | Easy to recognize and recall |
| **Meaningful** | Descriptive or persuasive content |
| **Likable** | Aesthetically appealing, fun to say |
| **Transferable** | Works across categories and geographies |
| **Adaptable** | Can evolve with the brand over time |
| **Protectable** | Legally and competitively defensible |

Total score = sum of all 6 (max 30). Higher is better.

## Step 6: Present Top 3 to User

For each of the top 3 candidates, present:
- The name and naming approach
- Keller scores (total and per criterion)
- Availability status (domain, social, trademark)
- Positioning fit rationale
- Linguistic notes (pronunciation, meaning)
- Recommended choice with clear reasoning

Let the user make the final selection. Document the choice and rationale.
