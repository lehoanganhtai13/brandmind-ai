# Deliverable Assembly — Phase 5

## Strategy Document Structure (10 Sections)

Assemble the brand strategy document (PDF/DOCX) with these sections:

1. **Executive Summary** — 1-page overview of the entire strategy
2. **Business Context** — Phase 0 diagnosis, problem statement, scope, budget tier
3. **Market Intelligence** — Phase 1 key findings, competitive landscape, target audience
4. **Strategic Synthesis** — SWOT, perceptual map, strategic sweet spot
5. **Brand Positioning** — Phase 2 positioning statement, POPs/PODs, value architecture
6. **Brand Identity** — Phase 3 personality, voice, visual direction, naming
7. **Communication Framework** — Phase 4 messaging hierarchy, channel strategy
8. **Implementation Roadmap** — 3 horizons with budget tier alignment
9. **KPI Framework** — Metrics, targets, measurement methods
10. **Appendices** — Mood boards, competitor profiles, research data

Delegate generation to the **document-generator** sub-agent with content from each phase output.

## Brand Key One-Pager

The Brand Key is a single-page visual summary of the brand strategy with 9 components:

| Component | Source |
|-----------|--------|
| **Root Strength** | Phase 0 business strengths + Phase 1 competitive advantage |
| **Competitive Environment** | Phase 1 competitive landscape summary |
| **Target** | Phase 1 target audience primary segment |
| **Insight** | Phase 1 top prioritized customer insight |
| **Benefits** | Phase 2 value architecture (functional + emotional) |
| **Values & Personality** | Phase 3 brand personality traits |
| **Reasons to Believe** | Phase 2 value architecture RTBs |
| **Discriminator** | Phase 2 key POD |
| **Brand Essence** | Phase 2 brand essence / mantra |

Delegate Brand Key visual generation to the **creative-studio** sub-agent with all 9 component texts.

## Presentation Deck Structure (10-12 Slides)

Assemble the pitch deck (PPTX) with these slides:

1. **Title slide** — Brand name + "Brand Strategy" + date
2. **The Challenge** — Business problem (Phase 0)
3. **Market Landscape** — Key findings (Phase 1)
4. **Target Audience** — Primary segment profile
5. **Brand Positioning** — Positioning statement + perceptual map
6. **Brand Identity** — Personality, voice, visual direction
7. **Brand Name & Logo** — Naming choice + visual concepts
8. **Communication Strategy** — Messaging + channels
9. **Implementation Roadmap** — 3-horizon visual timeline
10. **KPIs & Measurement** — Key metrics dashboard
11. **Investment Summary** — Budget breakdown
12. **Next Steps** — Immediate action items

Delegate generation to the **document-generator** sub-agent with structured content per slide.
